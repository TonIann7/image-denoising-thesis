# Metodi baseline di denoising

Questa cartella contiene i notebook relativi ai metodi baseline utilizzati
negli esperimenti della tesi.

I metodi considerati includono:

- filtro media;
- filtro mediano;
- BM3D;
- trasformata di Anscombe seguita da BM3D per il rumore di Poisson.

I notebook sono organizzati in base alla tipologia di rumore analizzata.
