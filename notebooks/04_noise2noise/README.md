# Noise2Noise

Questa cartella contiene i notebook relativi agli esperimenti
condotti con Noise2Noise.

I notebook includono:

- preparazione delle coppie di immagini rumorose;
- configurazione e addestramento del modello;
- applicazione del denoising;
- calcolo delle metriche PSNR e SSIM;
- salvataggio dei risultati sperimentali.
