# Noise2Void

Questa cartella contiene i notebook utilizzati per addestrare e valutare
Noise2Void sulle diverse tipologie di rumore considerate nella tesi.

I notebook includono:

- preparazione dei dataset rumorosi;
- estrazione delle patch;
- configurazione e addestramento del modello N2V;
- applicazione del denoising;
- calcolo delle metriche PSNR e SSIM;
- salvataggio dei risultati e dei file CSV.
